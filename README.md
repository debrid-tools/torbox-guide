# TorBox Guide

A practical guide to using a verified TorBox referral safely.

Read the live guide on the webpage:

- https://debrid-tools.github.io/torbox-guide/

## What this guide is for

This project is built to give you a referral code you can trust before your first TorBox purchase.
The page verifies the referrer's subscription status daily and shows whether the referral code is backed by an active paid plan.

## How to use it

1. Open the live guide above.
2. Confirm the status badge shows the referrer is on an active paid plan.
3. Use this referral code before completing your first checkout:

   `b6d7ded1-5aff-415b-b6d4-e73a50367859`
4. If paying with crypto, the coupon code `SIGMA30` can be applied for 30% off on one-time purchases only; it does not work for recurring subscriptions.

## Why it matters

- TorBox referral bonuses are only awarded on the first purchase.
- The referrer must have an active paid plan when you buy.
- If the referral is not valid, support will not retroactively add the bonus.

That means a random code from a forum is a real risk. This guide reduces that risk by checking the referrer daily via the TorBox API.
